# ReCipe-BalBesa
Инструкция по использованию скебобного приложения по поиску рецептов
1. Зайди в папку dist
2. найди и запусти приложение ReCipe-Balbesa.exe
3. в поисковую строку введи на английском ингредиенты по которым хочешь искать рецепт
4. нажми кнопку "Найти"
5. Ниже поисковой строки тебе выведутся рецепты с временем приготовления и состовляющими
6. Если хочешь найти ещё один рецепт нажми кнопку "Очистить"
7. Перед выходом из приложения нажми кнопку "Очистить"
8. Когда захочешь выйти нажми  на крестик в правом верхнем углу окна
